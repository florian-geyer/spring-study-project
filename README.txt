Учебный проект. Веб-приложение, на 4 страницы. Позволяет добавлять новые записи (прочитанные книги),
добавлять новых пользователей (которые добавляют новые книги), редактировать пользователей.

В проекте использовал Spring MVC, Spring Boot, Freemarker, Bootstrap.

Реализована отправка сообщений в веб-приложение (с распределением по БД), назначение ролей пользователей
(с сокрытием панели администратора от рядовых пользователей), загрузка файлов в приложение.

Будут реализованы редактирование и удаление данных из БД, логирование, интеграционное тестирование, с применением REST.

В pom добавлены jaxb-runtime и hibernate-core, чтобы проект запускался с Java 11
(Spring Boot не дружит пока с версиями Java старше JDK 1.8).
